# Project2-UE-ChallengeGame

Unreal Engine 5.7 挑战闯关游戏项目。

## 环境要求

- Unreal Engine 5.7+
- Visual Studio 2022 / Rider（C++ 开发）
- Git LFS（已配置，克隆前请确保已安装）

## 克隆

```bash
git lfs clone https://github.com/Yufang-cloud/Project2-UE-ChallengeGame.git
```

## 项目内容

- 场景关卡（Level_Scene_01）
- RPG 角色模型（Adventurer）及动画
- 多种材质、网格体、贴图资源
- 气球、木板、围栏等关卡互动元素

## 打开方式

1. 克隆仓库
2. 右键 `ChallengeGame.uproject` → **Generate Visual Studio project files**
3. 双击 `ChallengeGame.uproject` 打开引擎
